# GoShareFiles

GoShareFiles is a lightweight, web-based file sharing server written in Go. It provides a simple and efficient way to upload, download, and manage files and folders through HTTP endpoints.

## Features

- File Upload/Download
- Folder Creation and Management
- Repository System with Access Codes
- Automatic ZIP compression for folder downloads

## Installation

1. Clone the repository:
```bash
git clone https://github.com/NeedSuqqort/GoShareFiles.git
```

2. Navigate to the project directory:
```bash
cd GoShareFiles
```

3. Download dependencies:
```bash
go mod download
```

4. Run the application:
```bash
go run main.go
```

The application will start on port 8081 (http://localhost:8081)
